# 图片模型提示词插件 README

## 项目简介

这是一个为 Qwen、Z-Image 等图片生成模型设计的提示词管理插件。它集成了图片描述的所有常见分类，通过选择方式录入提示词，并收集了每个分类对应的常见值。插件支持自动保存、历史记录、导入导出功能，每个分类默认值为空但可选择随机生成，同时提供前置和后置手动输入区域。新增 NSFW 选项分类以满足不同内容需求。

## 主要功能

### 1. 分类化提示词管理
- 集成图片描述的所有常见分类（如风格、主题、构图、光照、色彩等）
- 每个分类提供预设的常见值选项供选择
- 分类默认值为空，避免强制填充

### 2. 智能录入方式
- 通过下拉选择、复选框等界面元素录入提示词
- 减少手动输入错误，提高提示词构建效率

### 3. 随机生成选项
- 每个分类保留“随机”选项
- 随机从该分类的常见值库中选择一个值
- 增加生成结果的多样性和探索性

### 4. 手动输入区域
- **前置提示词**：在分类提示词前添加自定义内容
- **后置提示词**：在分类提示词后添加自定义内容
- 保持灵活性，满足个性化需求

### 5. 数据管理功能
- **自动保存**：实时保存当前配置
- **历史记录**：记录和回溯之前的提示词组合
- **导入/导出**：支持 JSON 格式的配置导入导出
- 方便分享配置和在不同设备间同步

### 6. NSFW 内容选项
- 新增 NSFW（Not Safe For Work）分类选项
- 包含相关的内容级别和风格选择
- 明确标识，避免误用

## 安装与使用

### 安装方式
1. 下载插件文件到本地
2. 根据目标平台（如 Stable Diffusion WebUI、ComfyUI 等）的插件安装方式集成
3. 重启图片生成工具

### 基本使用流程
1. 打开插件界面
2. 根据需要选择各个分类的提示词值
3. 开启/关闭随机选项
4. 在前置/后置输入区域添加自定义内容
5. 点击生成，插件会自动组合所有提示词
6. 结果可保存到历史记录或导出为 JSON

### NSFW 选项使用说明
- 该选项默认设置为“none”（无 NSFW 内容）
- 如需生成相关内容，请明确选择相应级别
- 请确保使用环境允许生成此类内容
- 建议在合适的场景和受众下使用

## 配置说明

### 默认值设置
- 所有分类初始状态为空选择
- 用户必须主动选择或启用随机才会添加该分类提示词
- 避免生成不需要的分类词汇

### 随机选项逻辑
- 启用后，每次生成从该分类的选项库中随机选择一个
- 可与其他固定选择分类混合使用
- 支持批量启用多个分类的随机选项

### 提示词组合规则
1. 前置提示词（如存在）
2. 各分类选中的提示词（按固定顺序排列）
3. 后置提示词（如存在）
4. 所有部分用逗号分隔，形成最终提示词

## 导出/导入功能

### 导出配置
- 生成包含当前所有设置的 JSON 文件
- 包含分类选项、选择值、随机设置等
- 可用于备份或分享

### 导入配置
- 加载之前导出的 JSON 配置文件
- 恢复所有分类设置
- 历史记录可选择性导入

## 历史记录功能

- 自动记录最近 N 次生成的提示词组合（N 可配置）
- 显示生成时间和使用的分类
- 支持从历史记录中快速重新应用配置
- 可清除历史记录或导出为独立文件

## 自定义与扩展

### 添加新分类
1. 在分类配置文件中添加新分类
2. 定义分类名称和选项数组
3. 插件界面会自动显示新分类

### 修改选项库
1. 编辑对应分类的选项数组
2. 支持添加、删除或修改选项值
3. 更改后需要重新加载插件

### 调整界面布局
1. 通过修改 CSS/样式文件调整外观
2. 支持响应式设计适配不同屏幕
3. 可调整分类的显示顺序

## 注意事项

1. **兼容性**：确保插件版本与图片生成工具版本兼容
2. **性能**：分类和选项过多可能影响加载速度，建议合理组织
3. **存储**：历史记录和配置会占用本地存储空间
4. **更新**：定期检查分类选项库的更新，获取最新的提示词趋势
5. **NSFW 内容**：请负责任地使用该功能，遵守相关法律法规和平台政策
